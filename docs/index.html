Markdown Preview Enhanced

一个强大的 Markdown 预览插件，支持多种扩展语法和导出格式，适用于 VS Code、Atom 等编辑器。

安装命令：
npm install -g markdown-preview-enhanced

多格式导出

支持导出为 PDF、HTML、PNG、Markdown 等多种格式。

数学公式支持

内置 KaTeX/MathJax，轻松书写数学公式。

代码高亮

支持多语言代码高亮，适合技术文档编写。

幻灯片模式

一键切换 Markdown 幻灯片展示。
