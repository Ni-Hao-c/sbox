Markdown Preview Enhanced 仿制文档

本项目是对 markdown-preview-enhanced 说明文档风格的仿制。适用于 VS Code/Atom 等编辑器的 Markdown 预览插件，支持多种扩展语法和导出格式。

安装

npm install -g markdown-preview-enhanced

或在编辑器扩展市场搜索 Markdown Preview Enhanced 进行安装。

主要功能

  • 多格式导出：PDF、HTML、PNG、Markdown 等
  • 数学公式支持：KaTeX/MathJax
  • 代码高亮与图表支持
  • 幻灯片模式
  • 自定义样式与主题

用法示例

在 Markdown 文件中编写内容，点击“预览”即可实时查看效果。

```markdown
# 标题

- 支持数学公式：$E=mc^2$
- 支持流程图、甘特图等
```

FAQ

Q: 如何导出 PDF？

A: 在预览界面点击“导出”按钮，选择 PDF 格式即可。

Q: 支持哪些编辑器？

A: VS Code、Atom、Typora 等主流编辑器。
